CSS에 관하여

CSS 란

Cascading Style Sheets의 약자로, HTML, XHTML, XML 같은 문서를 꾸밀 때 사용하는 언어다.
기존에는 HTML언어 하나로 작성하여 HTML 문서를 수정할 때 모든 문서를 하나씩 수정해야 하는 번거로움이 있었다. 하지만 CSS의 등장으로 CSS 파일 하나만 수정하면 스타일에 해당하는 HTML 문서가 한번에 수정되는 엄청난 장점을 갖게 된다. 

	<!DOCTYPE html>
	<html>
	    <head>
	        <meta charset="utf-8"/>
	        <style type="text/css">
	            h1 {
	                color: red;
	                font-size: 20px;
	            }
	        </style>
	    </head>
	    <body>
	        <h1>CSS란?</h1>
	        <p>
	            html을 꾸며주는 아입니다.
	        </p>
	        <p>
	            Cascading Style Sheets 약자 이기도 하지요.
	        </p>
	    </body>
	</html>

CSS를 통해서 기대되는 효과들

  • 정보(HTML)과 디자인(CSS)를 분리할 수 있다.
  • 정보를 수정하지 않고 디자인만 변경할 수 있다.
  • 검색엔진이 HTML을 해석 가능하도록 하기 때문에 더 많은 방문자들이 방문하도록 유도할 수 있다.
  • HTML에서 디자인 분리함으로서 시각장애인을 위한 프로그램인 스크린리더가 HTML을 해석할 수 있도록 할 수 있다.
